ESSES SÃO MEUS PROJETOS:

nesta parte irei falar sobre projetos que desenvolvi e pretendo começar:

    • Website para Empresa Fictícia de Salgadinhos Desenvolvi em grupo, na faculdade, um website para uma empresa fictícia de salgadinhos, aplicando conceitos de design, programação e trabalho colaborativo. O projeto teve como objetivo criar um site moderno e responsivo, com foco em usabilidade, organização das informações e boa experiência para o usuário, simulando um cenário real de desenvolvimento profissional.

    • Sistema Backend com Banco de Dados SQL Desenvolvi um sistema backend completo integrado a um banco de dados relacional SQL. O projeto envolveu a modelagem e criação do banco de dados, definição de tabelas e relacionamentos, além da implementação de consultas e manipulação de informações. Foram aplicadas boas práticas de desenvolvimento para garantir organização, segurança e eficiência no gerenciamento dos dados.

    • Sistema Especialista – Restauro de Quadros Criei um Sistema Especialista (SE) voltado para auxiliar no processo de restauro de quadros. O sistema foi desenvolvido para simular a tomada de decisões de um profissional da área, oferecendo orientações com base em regras e conhecimentos técnicos previamente definidos. O projeto envolveu lógica de decisão, organização de conhecimento e aplicação prática de conceitos de Inteligência Artificial.
